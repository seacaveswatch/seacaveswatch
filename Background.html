Project Team Goals and Approach:

Project team partners have developed appropriate prototype software, radio communication details, completed field testing and designed an online web page for the system which allows real time wave measurements in front of the Mainland Sea Caves of the Apostle Islands. Project tests have shown that the system is viable and will allow park managers and the general public accessibility to real time wave information from locations miles away. This valuable information can be used to assist the park managers in management decisions to provide potential warnings of dangerous conditions which may occur at the Mainland Sea Caves. The ultimate goal is to have in place a hazard warning system which allows kayakers real time access via the web to valuable wave warning information (when waves may be too dangerous for kayaking) for the sea cave region which is only accessible by a mile and one half kayak trip.
